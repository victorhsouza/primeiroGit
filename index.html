Conceito de Div's

Definiçao:

Uma DIV esta relacionado a criar "DIVisoes" na pagina para que possamos posicionar os elementos na pagina HTML. Existem tags semanticas para as divs. Como por exemplo a tag header, main e footer utilizadas para dividir o conteudo da pagina.

Estamos começando a entender o conceito de div

Vamos criar uma outra div com outras formataçoes

Vantagens de DIVS

  • Maior praticidade ao distribuir elementos na pagina
  • Maior praticidade ao estilizar elementos
  • É divertido

Desvantagens de DIVS

  • Um pouco complexo o entendimento
  • É necessario esforço para aprender
  • Só depende de voce entender o funcionamento
